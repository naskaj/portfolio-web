Application of Ghost Module and SimAM in Guide Following for Smart Wheelchair based on YOLOv8N

This project focuses on developing a lightweight and efficient guide following system for smart wheelchair using a modified version of YOLOv8n, enhanced with Ghost Module and Simple Parameter-Free Attention Module. This system is designed to assist individuals with mobility challenges by allowing wheelchair to detect and follow a guide in real-time.
